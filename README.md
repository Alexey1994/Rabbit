# Rabbit
Это проект представляет собой некоторую часть 32 битной операционной системы для x86. Какие-либо работы по продолжению этого репозитария не предполагаются. Проект можно использовать в качестве помощника по написанию систем, код выполнен довольно чисто. ОС собирается и что-то может делать, но до логического конца не доведена. [Здесь](https://github.com/Alexey1994/programs/tree/master/%D0%B1%D0%B5%D0%BB%D0%BA%D0%B0) находится более интересное, полностью переписанное продолжение этого проекта. Но и оно не является конечным. Для себя пришёл к выводу что гораздо проще работать с системой через BIOS/UEFI чем писать все возможные драйвера для 32/64-битных систем, потому в данный момент пишу 32-х битную ОС для BIOS. Обещаю сделать красивое и понятное решение :)

This project represents some part of a 32-bit operating system for x86. No work is envisaged to continue this repository. The project can be used as an assistant to write systems, the code is executed quite cleanly. The OS is going and can do something, but it is not brought to its logical conclusion. [Here](https://github.com/Alexey1994/programs/tree/master/%D0%B1%D0%B5%D0%BB%D0%BA%D0%B0) is a more interesting, completely rewritten sequel to this project. But it is not finite. For myself, I came to the conclusion that it is much easier to work with the system via BIOS/UEFI than to write all possible drivers for 32/64-bit systems, so at the moment I am writing a 32-bit OS for the BIOS. I promise to make a beautiful and understandable solution :)
